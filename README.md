## 📦 ATools — горячие клавиши

Программа на AutoHotkey, которая добавляет удобные комбинации для Windows.

### ✨ Возможности

| Клавиши | Что делают |
|:---|:---|
| `Win + G` | Открыть Google |
| `Ctrl + Alt + T` | Вставить дату (ДД.ММ.ГГГГ) |
| `Win + ↑` | Громче |
| `Win + ↓` | Тише |
| `Win + M` | Выключить звук |
| `++--` | → `±` |

### 🚀 Как пользоваться

1. Скачай `mark.exe` из [релизов](https://github.com/Marrcus113/WinTools/releases/tag/at)
2. Запусти
3. Готово!

Если не работает — запусти **от имени администратора**.

### 🔧 Требования

- Windows 10 / 11
- Для запуска `.exe` ничего не нужно
